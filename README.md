# ArqGUIA

## O ArqGuia é uma plataforma para ajudar estudantes de Arquitetura e Urbanismo a organizarem seus estudos e não perderem nenhum detalhe enquanto fazem seus trabalhos e acompanham seu progresso!

### Funcionalidades

- Listas pré-montadas
- Ordem de prioridade
